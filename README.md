# 🌍 Sri Lanka Tourism Analytics Dashboard

## 📊 Project Overview
This project is an end-to-end data analytics solution built using Python and Microsoft Power BI. It analyzes Sri Lanka tourism data and provides interactive dashboards for business insights, KPIs, and trends.

The project includes:
- Python (Streamlit) interactive dashboard
- Power BI business intelligence dashboard
- Data cleaning & preprocessing using Python
- Visualization of tourism trends, revenue, and country insights
