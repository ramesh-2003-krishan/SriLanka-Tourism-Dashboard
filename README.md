# Sri Lanka Tourism Dashboard

A Power BI dashboard analyzing tourism patterns in Sri Lanka.

## Pages
- Overview — visitor counts, nationalities, yearly trends
- Most visited places — top destinations mapped across Sri Lanka
- Travel & transport — how tourists get around
- Accommodation & spending — stay types and top purchases
- Visitor insights — satisfaction ratings and tourism weaknesses

## Tools Used
Power BI Desktop · Excel · DAX

## Data Source
Sri Lanka Tourism Development Authority (SLTDA) annual reports
